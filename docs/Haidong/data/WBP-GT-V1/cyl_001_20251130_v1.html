S→G→M Three-Layer Assertion Framework

S - Structural Layer (S1-S20)

Checks IF something exists in the response. Independent, check first.

G - Grounding Layer (G2-G10)

Checks if VALUE matches source. Independent, each checks one value.

M - Meta Layer (M1)

No Hallucination check. Derived from all G results, computed last.
